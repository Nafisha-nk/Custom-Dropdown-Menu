# Custom Dropdown Menu (HTML, CSS & JavaScript)

A simple custom dropdown component built with plain HTML, CSS, and JavaScript.  
It has three states:
- **Default state** ‚Äì shows placeholder text.
- **Open state** ‚Äì reveals all available options.
- **Selected state** ‚Äì displays the chosen option and highlights it.
